# Adia - Luxury Jewelry Landing Page

Adia is a high-end luxury jewelry brand. This landing page was built to showcase the brand's bespoke craftsmanship, featuring immersive editorial design, responsive layouts, and modern typography.

## Features Built
- **Hero Section**: A stunning full-screen portrait with elegant typography.
- **Story Section**: An editorial-style layout detailing the brand's philosophy and craftsmanship.
- **Lookbook Section**: A horizontal scroll collage showcasing signature pieces.
- **Gemologist's Ledger (Featured Gems)**: An interactive accordion index that reveals detailed information about featured gems, paired with a gorgeous image collage.
- **Curated Acquisitions (Product Grid)**: A minimalist, hover-reveal product grid showcasing purchasable items.
- **Call to Action (CTA)**: An immersive, dark-themed section inviting users to "Begin Your Bespoke Journey."
- **Luxury Footer**: A classic 4-column footer with an elegant newsletter subscription form.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS v4
- **Typography**: Cormorant Garamond (Serif), Dancing Script (Script)
- **Deployment**: Ready for Vercel/Netlify

## Local Development
To run this project locally:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
